Blender学习笔记

发表于2023-01-07|更新于2023-01-09
|阅读量:

解释

  • “+”表示操作同时进行
  • “/“表示进行最前面操作后可进行操作
  • “ （此处为空格）”表示可选择其中一个

常用操作

视角移动

  • 俯视图：小键盘7
  • 仰视图：小键盘9
  • 左视图：小键盘1
  • 右视图：小键盘3
  • 鼠标中键：视角移动
  • Shift + 鼠标中键：水平移动
  • Ctrl + 鼠标中键：缩放

模型(点线面)操作

切换快捷键

  • 点：大键盘1
  • 线：大键盘2
  • 面：大键盘3

模型操作快捷键

  • 新建模型：shift + A
  • 模型移动：选中后 + G / G + x y z / G + x y z + 移动量
  • 模型旋转：选中后 + R / R + x y z / R + x y z + 旋转度数
  • 模型缩放：选中后 + S / S + x y z / S + x y z + 大小

挤出模式

选中点线面后 + E
移动点线面时不连带其他部分

切割

  • 常规切割：Ctrl + R + 左键确认操作 + 左键确认切割位置
  • 居中切割：Ctrl + R + 左键确认操作 + 右键居中切割

内插面

选择面后 + I
在选择的面内新增一个面

法向缩放

选中后 + Alt + S
缩放点线面时不改变点线面大小

无分类操作

3D游标设置：shift + 鼠标右键
3D游标复原：shift + C
进入编辑模式：Tab
透视模式：选中后 + Alt + Z

后期处理

纹理

渲染

摄像机

光源
